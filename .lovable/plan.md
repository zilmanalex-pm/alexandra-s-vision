

# Alexandra Zilman — Personal Portfolio Website

## Visual Foundation
- Light neutral background (#F8FAFF) with floating gradient blobs (purple/pink)
- Glassmorphism UI: frosted-glass cards with backdrop-blur, soft violet shadows
- Deep Violet (#6C5CE7) primary, Pink-Magenta (#FF7AD9) accents
- 24px rounded corners, smooth scroll, entrance fade animations, perspective tilt on card hover

## Section 1: Hero
- Bold headline: "Simplifying the complex. Delivering what matters."
- Sub-headline with name and specialization
- Two glassmorphism CTAs: "Download CV" (violet filled) and "LinkedIn" (ghost/outline)
- Animated floating abstract glowing brain/lightbulb element with purple aura (CSS/SVG animation)

## Section 2: About & Value Prop
- Two-column layout: circular headshot with glow on left, three icon-bullet value props on right
- Fade-in entrance animations on scroll

## Section 3: My Product Edge (Feature Grid)
- 3-column glass card grid with perspective tilt hover effect
- Cards: Comprehensive Vision, Product Tool Kit, Evidence ("The Receipts")
- Each card with icon, title, and bullet points

## Section 4: Case Study — TarbutON
- Interactive carousel for process steps (Mapping → Insights → Solutions → Roadmap)
- Side-by-side "Problem" vs "Solution" with device mockup frames
- Centered "Download Full Case Study" button with pulse animation

## Section 5: Testimonials (Wall of Love)
- Auto-scrolling horizontal ticker of testimonial cards
- Quotes from 6 people with avatar placeholders, names, and soft-bordered glass cards

## Section 6: Footer
- Minimalist centered layout with phone, email, location using Lucide icons
- Copyright line: "© 2026 Alexandra Zilman. Built for the future of SaaS."

## Technical Approach
- Single-page React app with smooth scroll navigation
- CSS animations for entrance fades, floating blobs, card tilts, and ticker scroll
- Fully responsive (mobile-first with breakpoints for tablet/desktop)
- Placeholder image for headshot, SVG/CSS for the hero visual element
